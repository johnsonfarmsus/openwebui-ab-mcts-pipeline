Idiot's Guide: Quick Install & Use

This is a super-simple, click-through guide to get everything running and connected.

Open Dashboard Open WebUI

1) Install prerequisites

  1. Install Docker Desktop
  2. Install Ollama and start it

2) Start everything

Open a terminal and run:

git clone <your-repo-url>
cd openwebui-setup
docker compose up -d

Wait ~1–2 minutes for services to start. Then open:

  • Dashboard: http://localhost:8081/dashboard.html
  • Open WebUI: http://localhost:3000

3) Set up Open WebUI connections (in the UI)

In Open WebUI, click the gear icon (bottom-left), then do these:

  1. Admin Settings → External Tools → add http://localhost:8097, click Verify Connection, then Save
  2. Settings → Connections → add http://localhost:8098, click Verify Connection, then Save
  3. Settings → External Tools → add http://localhost:8097, click Verify Connection, then Save
What are these?
  • 8098 = Model Integration (OpenAI-compatible) to select "models" like AB‑MCTS and Multi‑Model inside chat
  • 8097 = Tool endpoints (OpenAPI) for calling AB‑MCTS, Multi‑Model, and science helpers as tools

4) Try it

  1. Open WebUI → pick a model (e.g., "AB‑MCTS" or "Multi‑Model")
  2. Ask a question. For AB‑MCTS, expect a slower but deeper response.
  3. Visit Evals: /evals.html for charts and the tree view
  4. Open Grafana: http://localhost:3001 and switch to the AB‑MCTS dashboard

5) Common fixes

  • Nothing loads? Run docker compose ps to ensure all containers are Up
  • Prometheus/Grafana not loading? Wait 30–60 seconds after docker compose up -d
  • Open WebUI errors? Re-check the two connection URLs and click Verify Connection
